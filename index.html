O que é HTML

HTML significa HipeText Markup Language, ou seja, linguagem de marcação de hipertexto. Ela não é considerada uma linguagem de programação, mas sim, uma linguagem de marcação. Ela que dá o formato, a estrutura da página de internet.

Tags usadas para formatação de texto

Existem algumas tags que podem ser usadas nos textos para formatar a página em HTML, algumas principais como:

  1. Tag p: Para colocar em parágrafos, quando ela é usada, automaticamente é feita uma quebra de linha na página.
  2. Tag h1 até h6: São tags que dão níveis de seção ao texto, sendo h1 o mais alto e h6 o mais baixo.
  3. Tag b ou strong:Ambas deixam o texto em negrito, mas a strong tem um diferencial, de tornar o texto em pesquisas, com um ênfase maior durante a busca por meio de palavras chave
  4. Tag i: Itálico
  5. Tag u: Sublinhado
  6. Tag br: Quebra de linha

O que são atributos das tags

Em linha gerais, os atributos são informações adicionais que damos as Tags, ou seja, são características daquela tag, para que seja interpretados de maneiras diferente na página.

Sua sintaxe se dá dessa maneira: < tag atributo=”valor”> /tag>
